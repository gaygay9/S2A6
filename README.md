介紹
_________________________________________________
紀錄屬於自己的餐廳清單，可以瀏覽餐廳、查看詳細資訊。

功能
_________________________________________________
1. 查看所有餐廳
2. 瀏覽餐廳的詳細資訊
3. 搜尋特定餐廳
4. 新增餐廳
5. 編輯餐廳
6. 刪除餐廳

開始使用
_________________________________________________
請先確認有安裝 node.js 與 npm

將專案 clone 到本地

在本地開啟之後，透過終端機進入資料夾，輸入：

npm install
安裝完畢後，繼續輸入：

npm run start
若看見此行訊息則代表順利運行，打開瀏覽器進入到以下網址

Listening on http://localhost:3000
若欲暫停使用

ctrl + c

開發工具
_________________________________________________

Express 4.17.1

Express-Handlebars 5.3.4
